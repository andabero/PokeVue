# views
## Home
```
Se asignan los props a la clase PokemonCard haciendo uso de la llamada a la api con la clase Navigation.js.
Se asignan las filas y columnas de las PokemonCard a imprimir por pantalla.
Se crea un v-pagination para la navegacion en las distintas paginas.

```
## Pokemon
```
Mediante la evaluación de la URL se toma el id en ella y con ayuda de la clase Navigation.js se obtienen los datos del pokemon.
Con el uso de V-card se imprime las características del pokemon que se obtuvieron.
```
