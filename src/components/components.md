# components
## PokemonCard
```
Crea la tarjeta pokemon haciendo uso de la etiqueta v-card, esta se renderiza cuando se reciben las propiedades (props) que se le asignan desde Home.vue.
La imagen y el boton "ver mas" asignan un valor en la URL donde se ejecuta la clase Pokemon.vue
```
## AppBar
```
Crea una barra estatica que tiene la funcionalidad de buscar pokemon por id/numero, esto lo logra cambiando la URL donde se ejecuta la clase Pokemon.vue
```
